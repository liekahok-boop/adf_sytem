==== LANGKAH 1: CLEAR BROWSER CACHE ====
1. Buka aplikasi di browser (http://localhost:8080/adf_system/)
2. Tekan: Ctrl + Shift + Delete (Windows) atau Cmd + Shift + Delete (Mac)
3. Pilih "Cached images and files"
4. Klik "Clear Data" atau "Clear Browsing Data"

==== LANGKAH 2: HARD REFRESH ====
1. Tekan: Ctrl + F5 (Windows) atau Cmd + Shift + R (Mac)
2. Tunggu halaman load ulang
3. Buka Browser Console: F12

==== LANGKAH 3: TEST ENDPOINT BARU ====
API lama: /api/end-shift.php (mungkin cache)
API BARU: /api/end-shift-new.php (CLEAN - NO CACHE)

JavaScript sudah diupdate untuk gunakan endpoint baru!

==== LANGKAH 4: CLICK END SHIFT ====
1. Klik tombol "End Shift" (pink button)
2. Lihat Console (F12) untuk debug messages
3. Seharusnya modal muncul dengan report data

==== JIKA MASIH ERROR ====
1. Laporkan screenshot console error
2. Saya akan debug lebih detail

PENTING: 
- Jangan gunakan cache/offline mode
- Ensure anda sudah login
- Check console untuk error details
